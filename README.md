# SpaceX Data Science Capstone Project

This repository contains the complete implementation of the IBM Data Science Capstone Project.

## Project Objective
To predict the success of Falcon 9 first stage landings using historical SpaceX launch data.

## Data Sources
- SpaceX REST API
- Wikipedia (Web Scraping)

## Project Structure
- notebooks/: Jupyter notebooks for data collection, EDA, SQL, visualization, and ML
- dashboard/: Plotly Dash application
- images/: Screenshots used in presentation

## Methodology
- Data Collection using SpaceX API and Web Scraping
- Data Wrangling and Feature Engineering
- Exploratory Data Analysis using Visualization and SQL
- Interactive Visual Analytics using Folium and Plotly Dash
- Predictive Analysis using Classification Models

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Decision Tree
- K-Nearest Neighbors (KNN)

## Best Result
Decision Tree classifier achieved the highest accuracy of 83.33%.
